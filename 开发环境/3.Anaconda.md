## 安装

```
wget -c https://repo.anaconda.com/archive/Anaconda3-2025.06-0-Linux-x86_64.sh

bash Anaconda3-2025.06-0-Linux-x86_64.sh

printf '\n# Anaconda path\nexport PATH=~/anaconda3/bin:$PATH\n' >> ~/.bashrc

source ~/anaconda3/bin/activate
source ~/.bashrc

conda --version
```

## 创建python环境

```
conda create -n llama-factory-rocm python=3.12

conda activate llama-factory-rocm

conda env list
```

## 删除python环境

```
conda remove -n llama-factory-rocm --all
```
